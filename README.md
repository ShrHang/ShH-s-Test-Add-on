## Introduction

This add-on implements 4 custom commands:

- `/shh:menu`：Open the menu
- `/shh:openmenu [player]`：Only operator, open the menu for a specified player
- `/shh:tppos <location>`：Teleport to specified coordinates
- `/shh:tpentity <entity>`：Teleport to a specified entity

Except for special instructions, all of the above commands can be used directly by players in the game.

The command to open the menu will display a simple UI interface on the player's screen, which can be used to teleport to waypoints and switch game modes.  
The waypoint teleportation function will teleport the player to preset coordinate locations, which can be modified in the [`tpPoints.ts`](scripts/tpPoints.ts) file.  

## Brief Explanation of the Build Method

The build method for this add-on is consistent with the official developer pack sample project, and the main steps are as follows:

1. Clone the project to your local machine.
2. Run the command to install dependencies in the root directory of the project:
  ```powershell
    npm i
  ```
3. Run the build command in the root directory of the project:
  ```powershell
    npm run build
  ```
4. Import the contents of the `dist/package/` folder generated by the build into Minecraft.
